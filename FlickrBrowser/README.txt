PROJECT: Flickr Browser

Description
----------
This project is an updated/revised App project on top of the teaching project,
FlickrBrowser, from Udemy Android courses.
  - link: https://www.udemy.com/android-marshmallow-java-app-development-course/learn/v4/overview
  - section 9
 This updated version replaces Picasso with SQLite.


Known Bugs
----------
1. Online search results are persisted in the DB but are not shown in view.
2. Same search results are persisted in DB.
3. App crashes if DB holds many records.
4. View list only display the first 20 records from DB.
